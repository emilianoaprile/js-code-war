# Repeat String

## Descrizione
In questo esercizio, devi scrivere una funzione che accetta un numero intero `n` e una stringa `s` come parametri e restituisce una stringa `s` ripetuta esattamente `n` volte.

## Input e Output

- L'input della funzione è un numero intero `n` e una stringa `s`.
- L'output della funzione è una stringa che rappresenta la stringa `s` ripetuta `n` volte.

## Esempio

- 6, "I"     -> "IIIIII"
- 5, "Hello" -> "HelloHelloHelloHelloHello"
