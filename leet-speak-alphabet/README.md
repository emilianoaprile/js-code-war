# Cypher - Crittografia

## Descrizione
Questo esercizio riguarda la crittografia di una stringa di testo basata su una mappatura di caratteri specifica. Ricevi una stringa composta da parole inglesi e restituisci una versione cifrata di quella stringa. La mappatura di caratteri Ã¨ la seguente:

| Numero | Maiuscolo | Minuscolo |
| ------ | --------- | --------- |
| 1      | I         | l         |
| 2      | R         | z         |
| 3      | E         | e         |
| 4      | A         | a         |
| 5      | S         | s         |
| 6      | G         | b         |
| 7      | T         | t         |
| 8      | B         | -         |
| 9      | -         | g         |
| 0      | O         | o         |

Tutti gli altri caratteri fuori dalla tabella rimangono invariati quando cifrati.

## Esempi

- Input: "Hello World"
  Output: "H3110 W0r1d"

- Input: "I am your father"
  Output: "1 4m y0ur f47h3r"

- Input: "I do not know what else I can test. Be cool. Good luck"
  Output: "1 d0 n07 kn0w wh47 3153 1 c4n 7357. 83 c001. 600d 1uck"
